Project Description
This project is a basic Simple E-commerce Store developed as part of the Full Stack Development Internship task.
The application displays products, allows users to add items to a shopping cart, and view the cart.
It demonstrates the basic concepts of frontend web development and simple cart functionality.
Features
Product listing page
Add products to cart
Shopping cart page
Basic user interface using CSS
Cart storage using Local Storage
Technologies Used
HTML
CSS
JavaScript
